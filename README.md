# AI Predictive Maintenance System

## Description
This project predicts machine failure using machine learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Features
- Generates synthetic sensor data
- Trains Random Forest model
- Predicts equipment failure


## How to Run
1. Install required libraries
2. Run main.py file
